# Task Automation Script using Python

This is a simple Python automation project developed as an internship task.

## Project Objective
Automatically extract email addresses from a text file and save them into another file.

## Features
- Reads data from a text file
- Extracts email addresses using Regular Expressions
- Saves extracted emails into another file
- Displays extracted emails on the console

## Concepts Used
- Python
- Regular Expressions (re module)
- File Handling
- Loops

## Files Used
- sample.txt
- emails.txt
- automation_script.py

## How to Run
1. Open the Python file
2. Run the program
3. Extracted emails will be saved automatically
